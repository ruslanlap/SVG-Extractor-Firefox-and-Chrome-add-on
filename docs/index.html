Privacy Policy for SVG Extractor Pro

Last Updated: November 21, 2025

Introduction

SVG Extractor Pro is a browser extension designed to help users find, extract, and download SVG images from web pages. This privacy policy explains what data the extension collects, how it's used, and your rights regarding your data.

Data Collection and Use

What Data We Collect

SVG Extractor Pro collects and processes only the following data:

  1. SVG Images
    • The extension scans web pages for SVG images when you click the extension icon
    • SVG content is temporarily stored in your browser's local storage
    • This data is used solely to display results and enable downloads
    • SVG data is cleared when you close the results page or restart your browser
  2. User Preferences
    • Theme preference (light/dark mode)
    • Language preference (English/Ukrainian)
    • These settings are stored locally in your browser using Chrome's storage API

What We DON'T Collect

  • We do NOT collect any personal information
  • We do NOT collect browsing history
  • We do NOT track which websites you visit
  • We do NOT collect any user credentials or passwords
  • We do NOT use analytics or tracking tools
  • We do NOT collect or transmit any data to external servers

Data Storage

All data collected by SVG Extractor Pro is stored locally on your device using Chrome's storage API. We do not have access to this data, and it never leaves your device.

Data Sharing

We do not share, sell, or transfer any data to third parties. All data processing happens entirely on your local device.

Permissions Explained

SVG Extractor Pro requests the following permissions to function:

  • activeTab: Access the current tab's content to scan for SVG images
  • storage: Store user preferences (theme, language) and temporary SVG data locally
  • downloads: Enable downloading of SVG files to your computer
  • tabs: Manage the results tab (create, update, and check if it exists)
  • scripting: Inject content scripts to scan web pages for SVGs
  • host_permissions (<all_urls>): Access any website's content to extract SVG images, as users may want to extract SVGs from any website they visit

Third-Party Libraries

SVG Extractor Pro uses the following third-party library:

  • JSZip (jszip.min.js): Used to create ZIP files when downloading multiple SVGs. This library runs entirely in your browser and does not transmit any data externally.

Data Security

  • All data processing happens locally in your browser
  • No data is transmitted over the network to our servers (we don't have any servers)
  • The extension uses Chrome's Content Security Policy to prevent unauthorized code execution

User Rights

You have the right to:

  • Access: View all data stored by the extension in Chrome's developer tools
  • Delete: Remove the extension at any time, which will delete all stored data
  • Control: Choose which websites to use the extension on

Compliance with Chrome Web Store Policies

SVG Extractor Pro complies with all Chrome Web Store Developer Program Policies:

  • Single Purpose: The extension has one clear purpose - to extract and download SVG images
  • Limited Use: Data is used only for the extension's stated purpose
  • No Remote Code: The extension does not load or execute remote code (Manifest V3 compliant)
  • Minimal Permissions: Only necessary permissions are requested

Limited Use Disclosure

The use of information received from Google APIs adheres to the Chrome Web Store User Data Policy, including the Limited Use requirements.

Children's Privacy

SVG Extractor Pro does not knowingly collect any information from children. The extension is intended for general use and does not require any personal information.

Changes to This Policy

We may update this privacy policy from time to time. The "Last Updated" date at the top of this document indicates when the policy was last revised. Continued use of the extension after any changes constitutes acceptance of the updated policy.
