# 🚀 Швидка інструкція встановлення SVG Extractor Pro для Chrome

## 📦 Крок 1: Підготовка

У вас є 2 варіанти:

### Варіант А: Використати папку `chrome-version`
Просто відкрийте папку `chrome-version` на вашому комп'ютері

### Варіант Б: Розпакувати ZIP архів
1. Знайдіть файл `SVG-Extractor-Pro-Chrome-v1.1.1.zip`
2. Розпакуйте його в будь-яку папку
3. Відкрийте розпаковану папку `chrome-version`

---

## 🌐 Крок 2: Відкрийте налаштування розширень Chrome

1. Запустіть браузер **Google Chrome**
2. В адресному рядку введіть: **`chrome://extensions/`**
3. Натисніть **Enter**

> 💡 **Підказка**: Це спеціальна сторінка Chrome для керування розширеннями

---

## 🔧 Крок 3: Увімкніть режим розробника

1. На сторінці розширень знайдіть **перемикач у правому верхньому куті**
2. Він називається **"Режим розробника"** або **"Developer mode"**
3. **Клікніть на перемикач**, щоб увімкнути його
4. Перемикач стане **синім** або переміститься вправо

```
┌─────────────────────────────────────────┐
│  Розширення              [Режим розробника] ← Увімкніть це
│  Extensions              [Developer mode]│
└─────────────────────────────────────────┘
```

---

## 📂 Крок 4: Завантажте розширення

1. Після увімкнення режиму розробника **з'являться нові кнопки зверху**
2. Знайдіть і натисніть кнопку:
   - **"Завантажити розпаковане розширення"**
   - або **"Load unpacked"** (англійською)

3. **Відкриється вікно вибору папки**

4. Знайдіть і **виберіть папку `chrome-version`**
   - Це папка, де знаходяться файли розширення
   - Там мають бути файли: manifest.json, background.js тощо

5. Натисніть **"Вибрати папку"** або **"Select Folder"**

```
┌─────────────────────────────────────────┐
│  [Завантажити розпаковане розширення]   │ ← Натисніть
│  [Load unpacked]                        │
└─────────────────────────────────────────┘
```

---

## ✅ Крок 5: Готово!

Якщо все пройшло успішно:

1. Ви побачите **нову картку розширення** "SVG Extractor Pro"
2. З'явиться **іконка розширення** на панелі інструментів Chrome (біля адресного рядка)
3. Розширення готове до використання!

---

## 🎯 Як користуватися

### Простий спосіб:
1. Відкрийте **будь-який сайт** (наприклад, wikipedia.org)
2. **Натисніть на іконку** SVG Extractor Pro на панелі інструментів
3. **Відкриється нова вкладка** з результатами
4. Тепер ви можете:
   - 💾 **Завантажити** окремі SVG
   - 📋 **Скопіювати** код SVG
   - 🖼️ **Зберегти як PNG**
   - 📦 **Завантажити всі SVG** одразу

## ❓ Поширені питання

### ❌ Розширення не встановлюється
**Перевірте:**
- Чи увімкнений "Режим розробника"?
- Чи вибрали ви правильну папку `chrome-version`?
- Чи містить папка файл `manifest.json`?

### ❌ Не бачу іконки розширення
**Рішення:**
- Клікніть на іконку пазла (🧩) на панелі інструментів
- Знайдіть "SVG Extractor Pro"
- Натисніть на значок кнопки (📌), щоб закріпити іконку

### ❌ Розширення не працює на деякіх сайтах
**Це нормально!**
Chrome не дозволяє розширенням працювати на:
- Внутрішніх сторінках (`chrome://`, `edge://`)
- Сторінках налаштувань
- Сторінках магазину розширень

### ⚠️ Chrome показує попередження
**Не хвилюйтесь!**
Це стандартне попередження для розпакованих розширень.
Розширення безпечне - ви можете переглянути весь код.

---

## 🔄 Як оновити розширення

1. Замініть файли в папці `chrome-version` новими
2. Перейдіть на `chrome://extensions/`
3. Знайдіть SVG Extractor Pro
4. Натисніть кнопку **оновлення** (🔄)

---

## 🗑️ Як видалити розширення

1. Перейдіть на `chrome://extensions/`
2. Знайдіть SVG Extractor Pro
3. Натисніть **"Видалити"** або **"Remove"**
4. Підтвердіть видалення
